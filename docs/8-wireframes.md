| 동작 | 설명 |
|------|------|

| 비밀번호 확인 | 비밀번호와 불일치 시 인라인 오류 표시 |
| 가입하기 클릭 | 전체 유효성 통과 시 API 호출. 이메일 중복(HTTP 409) 시 "이미 사용 중인 이메일입니다." 오류 표시 |
| 가입 성공 | 기본 카테고리 자동 생성 후 S-02 로그인 화면으로 자동 이동 |
| 로그인 링크 클릭 | S-02 로그인 화면으로 이동 |

### 5. 인터랙션 설명

| 동작 | 설명 |
|------|------|
| 전체 탭 클릭 | UC-012: 로그인 사용자의 모든 할일 목록 조회 |
| 카테고리 탭 클릭 | UC-013: 카테고리 드롭다운 노출. 카테고리 선택 시 해당 카테고리 할일만 표시 |
| 시작 전 탭 클릭 | UC-014: isCompleted = false AND 오늘 < startDate 조건의 할일 표시 |
| 진행 중 탭 클릭 | UC-015: isCompleted = false AND startDate <= 오늘 <= endDate 조건의 할일 표시 |
| 완료 탭 클릭 | UC-016: isCompleted = true 조건의 할일 표시 |
| 기한 초과 탭 클릭 | UC-017: isCompleted = false AND 오늘 > endDate 조건의 할일 표시 |
| 완료 토글 클릭 | UC-011: isCompleted 토글. 완료 취소 시 날짜 기반 상태 재계산. 목록 즉시 갱신 |
| 수정 버튼 클릭 | S-04 할일 수정 화면으로 이동. 해당 할일 데이터 전달 |
| 삭제 버튼 클릭 | UC-010: 삭제 확인 모달 표시. 확인 시 API 호출 후 목록에서 즉시 제거 |

| 상태 | 기호 | 의미 |
